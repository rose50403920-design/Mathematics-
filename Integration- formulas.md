# Integration Formulas - Complete Cheat Sheet

## 1. Basic Formulas
**∫ 1 dx = x + C**  
**∫ xⁿ dx = x^(n+1)/(n+1) + C**  [n ≠ -1]  
**∫ 1/x dx = ln|x| + C**  
**∫ e^x dx = e^x + C**  
**∫ a^x dx = a^x / ln(a) + C**

## 2. Trigonometric
**∫ sin(x) dx = -cos(x) + C**  
**∫ cos(x) dx = sin(x) + C**  
**∫ tan(x) dx = -ln|cos(x)| + C**  
**∫ cot(x) dx = ln|sin(x)| + C**  
**∫ sec(x) dx = ln|sec(x) + tan(x)| + C**  
**∫ csc(x) dx = -ln|csc(x) + cot(x)| + C**  
**∫ sec²(x) dx = tan(x) + C**  
**∫ csc²(x) dx = -cot(x) + C**  
**∫ sec(x)tan(x) dx = sec(x) + C**  
**∫ csc(x)cot(x) dx = -csc(x) + C**

## 3. Inverse Trigonometric
**∫ 1/√(1-x²) dx = sin⁻¹(x) + C = -cos⁻¹(x) + C**  
**∫ 1/(1+x²) dx = tan⁻¹(x) + C = -cot⁻¹(x) + C**  
**∫ 1/(|x|√(x²-1)) dx = sec⁻¹(x) + C = -csc⁻¹(x) + C**

## 4. Logarithmic & Exponential
**∫ ln(x) dx = x ln(x) - x + C**  
**∫ logₐ(x) dx = x logₐ(x) - x/ln(a) + C**

## 5. Special Forms
**∫ 1/(x²+a²) dx = (1/a) tan⁻¹(x/a) + C**  
**∫ 1/(x²-a²) dx = (1/2a) ln|(x-a)/(x+a)| + C**  
**∫ 1/√(x²+a²) dx = ln|x + √(x²+a²)| + C**  
**∫ 1/√(x²-a²) dx = ln|x + √(x²-a²)| + C**  
**∫ √(x²+a²) dx = (x/2)√(x²+a²) + (a²/2)ln|x + √(x²+a²)| + C**  
**∫ √(x²-a²) dx = (x/2)√(x²-a²) - (a²/2)ln|x + √(x²-a²)| + C**

*Note: C = constant of integration*
